FortiGate-SyntaxChecker Help

Overview

FortiGate-SyntaxChecker is a Python script that checks the CLI syntax of FortiGate backup files. It helps you find common mistakes and formatting issues before restoring or uploading configurations.

Installation

  1. Download or clone the repository to your computer.
  2. Ensure you have Python 3.7 or newer installed.
  3. Open a terminal or command prompt in the project folder.
  4. Install the required package by running:
    pip install -r requirements.txt

Usage

To check a FortiGate CLI script file, run:

python syntax-checker.py path/to/cli_script.txt

To include warnings in the output, add the --warnings option:

python syntax-checker.py path/to/cli_script.txt --warnings

Output

The script prints a table of potential issues found in your CLI script, including:

  • Line number
  • Severity (ERROR, WARNING, INFO)
  • Issue description
  • Details and context

Notes

  • This tool uses heuristic checks and is not an authoritative parser for FortiGate CLI.
  • It is intended to help catch common mistakes and formatting issues before uploading or restoring configurations.
